### 项目名称：

        基于系统调用重载的系统级资源访问审计

### 项目团队

        麻家乐 王梓睿 孙靖轩 张天烁 汪何希

### 主要工作

        1. 获取系统调用表 sys_call_table。
        2. 实现系统函数重载和地址写入。
        3. 实现内核和用户层间的信息交互。
        4. 将信息保存为日志及尝试相关扩展处理。
        5. 实现了图形化界面，提供可视化操作方便用户使用。

### 开发环境

        Ubuntu 22.04  linux-5.19
